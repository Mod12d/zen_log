# 動機
- 情報があまりないが、MutableListで二重配列を使用したかった
- MutableListの二重配列で値を代入すること、その際に詰まりそうなことを情報として展開したい

# 事象
- 一次元のリストを代入後、二次元リストがループの最後の一次元のリストの値で、全ての二次元配列の要素が更新されていた

# 解決策
- ループ内で一次元のリストを毎回定義する
